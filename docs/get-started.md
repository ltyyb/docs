# 审核流程

本音游部采用问卷审核形式，任何 厦门六中同安校区 的学生均可通过自助填写问卷申请加入本社群。

:::warning
审核流程以及相关条约可能会根据实际情况进行调整，请参考最新群公告以及本手册内容。
:::

## 阅读条款

在申请加入本群之前，请务必仔细阅读并理解以下条款：
